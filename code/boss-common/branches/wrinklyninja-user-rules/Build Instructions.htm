BOSS - Better Oblivion Sorting Software: Build Instructions

This guide presumes that you are building on Windows. If not, presumably you are experienced enough to know what you're doing. If after following this guide, you are still having trouble compiling BOSS, feel free to PM pacificmorrowind or WrinklyNinja for help.

This guide also assumes that you are building the 'trunk' of the boss-common code. If you are building another trunk or a branch, change all references to boss-common\trunk\ to the directory that contains the BOSS.sln file of the code you are building.


Preparing Your Environment/Getting Prerequisites

Download and install:

  • Microsoft Visual C++ 2010 Express Edition
  • Windows SDK - You only need to install the "x86 Libraries" and "Windows Headers" from the "Windows Headers And Libraries" section.
  • Curl Source Code - Download the latest zipped source code from the top of the page.
  • BOOST C++ libraries
  • An SVN client of your choice.

Preparing BOSS and BOOST

  1. Perform a SVN Checkout from https://better-oblivion-sorting-software.googlecode.com/svn to a local directory of your own choice to create your working copy. (This directory shall be referred to as $BOSS from here on in.)
  2. Extract the contents of the boost-x.xx.x (x is a number) folder in the BOOST libaries archive you downloaded to $BOSS\code\boss-common\trunk\boost\ (you should have lib, include, boostrap.bat, etc. in this folder).
  3. Start a Command Prompt and cd to $BOSS\code\boss-common\trunk\boost and run bootstrap.bat
  4. In the Command Prompt, run bjam --toolset=msvc-10.0 --build-type=complete . This will take a while. Once that is done, you may close the Command Prompt window.

Preparing Curl dependency

We are going to be building the curl requirements from source, since the Windows curl packages are often not up-to-date. We don't need to compile all of curl though, just the static library. To do so, follow these steps:

  1. Extract the contents of the curl-x.xx.x (x is a number) folder in the curl archive to $BOSS\code\boss-common\trunk\curl\ (you should have lib, include, README, etc. in this folder).
  2. Open $BOSS\code\boss-common\trunk\curl\lib\libcurl.vcproj in Visual C++ 2010. When the Conversion Wizard window appears, just click 'Finish' then 'Close'.
  3. Set the build type to Release. In Project->Properties->Configuration Properties->C/C++->Code Generation, set the Runtime Library to "Multi-threaded (/MT)".
  4. Build the libcurl project. This will create a libcurl.lib file in the lib/Release directory. An 'Unable to start' error message will pop up, ignore this and click 'OK'. Close the libcurl project.


Building BOSS

  1. Open $BOSS\code\boss-common\trunk\BOSS.sln in Visual C++ 2010.
  2. Set the build type to "Release" and run Build on the Common project, which will give you BOSS.exe and supports Oblivion, Fallout 3 and Morrowind. Alternatively, run Debug->Build Solution to build all the projects in the solution.
  3. The generated exe(s) will be located in $BOSS\code\boss-common\trunk\%BUILDTYPE%\. If you are building for Fallout 3, you can rename the BOSS.exe to BOSS-F.exe if you wish.
